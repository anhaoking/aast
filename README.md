# 爱因斯坦棋 (Einstein Chess)

基于 Node.js 的爱因斯坦棋对弈平台，支持人机对战、本地双人对战和在线联机对战。

## 功能特性

- 🎮 **三种对战模式**：人机对战、本地人人对战、在线联机对战
- 🔴🔵 **阵营分配**：服务器随机分配红蓝双方
- 🎲 **骰子驱动**：掷骰决定可移动的棋子
- 📋 **棋谱导出**：支持导出完整对局棋谱
- 📚 **历史棋谱**：查询和回放已完结的历史对局
- 👥 **用户系统**：支持注册登录和游客模式
- 🔄 **再来一局**：联机模式下双方确认后可快速重开

## 技术栈

- **前端**：原生 HTML/CSS/JS，Canvas 棋盘渲染
- **后端**：Node.js + Express + Socket.IO
- **存储**：JSON 文件持久化（用户数据、棋谱记录）

## 快速开始

### 环境要求

- Node.js >= 14.x

### 安装与运行

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 或双击运行
start.bat
```

启动后访问 `http://localhost:3000` 即可开始游戏。

## 游戏规则

- 5×5 棋盘，红蓝双方各有 6 枚棋子（编号 1~6）
- 双方骰子点数 1~6，掷出几就只能移动对应编号的棋子
- 棋子移动方向由阵营决定（红方向右/右下/下，蓝方向左/左上/上）
- 目标：将自己的棋子全部移动到对方起始区域
- 无棋可走或全部到达目标区即告结束
